BOOTSTRAP 3 GRIDS

Designers and Developers for a Device Agnostic Internet.

By: Matthew Spangenberg

An Introduction

As the world of devices and resolutions continue to vastly expand, it is becoming increasingly difficult to translate design into code and code into design. How can a designer start with a fixed document size in Photoshop when the medium of web is unbound by a fixed size? How can a developer accurately assemble a design when it is created with an optimal single space in mind by a designer?

This made my skin crawl and brain ache. Being a huge advocate of efficiency, there had to be a better way than designing numerous layouts and translating them to code. Finally, I found a solution to this ever growing problem. It was leveraging a framework called Bootstrap. Bootstrap (3 at time of writing) is designed to take a website's layout and gracefully translate it anywhere from a large screen to an extra small screen. The framework introduces four 12 column layouts for different resolutions to solve for this lack of control.

Bootstrap addresses a team's need to rethink how they create experiences using a mutual language that can be understood by both designers and developers alike. It may not be the ultimate solution, but it is a great starting point.
